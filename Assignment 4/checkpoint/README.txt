README
---------
Environment - Contains images of our physical enviorment and the replicated enviorment in MATLAB
 - environment# - images of the physical enviorment
 - Matlab screenshot - a screenshot of the MATLAB figure with the plotted landmarks and the robot
 
Nonclosure - contains logged data and images from the run through the environment to test basic SLAM functionality
Closure - contains logged data and images from the run through the environment intended to test loop closure

For both Closure and Nonclosure:
 - The AVI file contains a sped up, low quality video of our run through the enviorment
 - the log text file contains the sensor data from each run
 - logData python file contains the motion plan used for each run
 - pic-XXXXXXXX contians the logged images for each landmark in the run
